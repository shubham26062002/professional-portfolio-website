## Installation

Clone the repository

```bash
git clone https://github.com/shubham26062002/professional-portfolio-website.git
```

Install packages

```bash
bun install
```

Setup environment variables

`NEXT_PUBLIC_MY_EMAIL`

`NEXT_PUBLIC_MY_NAME`

`RESEND_SENDER_EMAIL`

`RESEND_API_KEY`

```bash
bun run dev
```
